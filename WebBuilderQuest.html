HTML Levels

  1. HTML Basics

    Learn the foundational elements of HTML (HyperText Markup Language), including tags and attributes used to structure and define content on web pages.

  2. Lists and Links

    Explore how to create ordered and unordered lists in HTML, and how to use anchor a tags to create hyperlinks between web pages and resources.

  3. Forms and Inputs

    Understand how to create forms in HTML and utilize various input elements input, textarea, select to gather user input and data.

  4. Images and Alt Text

    Learn how to integrate images img into web pages and the importance of providing descriptive alternative text (alt) for accessibility.

  5. Tables

    Explore the table element and its related tags tr, td, th to structure tabular data effectively within web pages.

  6. Semantic HTML

    Discover the concept of semantic HTML, using elements like header, nav, section, article, footer, and aside to enhance the structure and meaning of web content for improved accessibility.

CSS Levels

  1. Basic Styling

    Learn fundamental CSS properties and selectors to style HTML elements, including basic text formatting, background colors, and borders.

  2. Selectors and Colors

    Explore CSS selectors in depth, including class, ID, attribute, and pseudo-class selectors, along with techniques for defining colors using HEX, RGB, and named colors.

  3. Box Model and Margins

    Understand the CSS box model, which includes margins, borders, padding, and content dimensions, to control the layout and spacing of elements.

  4. Flexbox Layout

    Master the Flexbox layout model for creating flexible and responsive layouts, enabling alignment, ordering, and distribution of elements within containers.

  5. Grid Layout

    Dive into the CSS Grid layout system to build complex two-dimensional layouts, providing precise control over rows, columns, and grid items.

  6. Responsive Design with Media Queries

    Learn how to create responsive web designs that adapt to different screen sizes and devices using CSS media queries and viewport units.

  7. Transitions and Animations

    Explore CSS transitions and animations to add dynamic effects to elements, including transformations, fades, and movement on user interaction.

  8. Positioning and Z-index

    Master CSS positioning techniques such as relative, absolute, fixed, and sticky positioning, along with managing stacking order using z-index.

  9. Styling Links and Lists

    Enhance user experience by styling links a tags and lists ul, ol, li with CSS, including hover effects, custom bullet points, and list styling.

  10. CSS Variables

    Utilize CSS variables (custom properties) to centralize and manage design constants like colors, spacing, and fonts, enhancing maintainability and flexibility in stylesheets.

Javascript Levels

  1. Basic Output

    Learn how to output content to the browser console and web page using console.log() and DOM manipulation methods.

  2. Variables and Types

    Understand JavaScript variables, data types (string, number, boolean, etc.), and how to declare, assign, and manipulate values.

  3. Functions

    Explore JavaScript functions, including function declarations, parameters, return values, and how to invoke functions to perform tasks.

  4. Conditional Statements

    Master conditional statements (if, else if, else) in JavaScript to control the flow of execution based on different conditions.

  5. For Loops

    Learn how to use for loops to iterate over arrays and perform repetitive tasks with JavaScript, including iterating backwards and skipping iterations.

  6. While Loops

    Understand the while and do...while loop structures in JavaScript for executing code repeatedly until a specified condition is no longer true.

  7. Arrays

    Explore JavaScript arrays, including creating arrays, accessing elements by index, adding and removing elements, and common array methods like push, pop, slice, and forEach.

  8. Objects

    Dive into JavaScript objects, including object literals, properties, methods, and how to access and manipulate object data using dot notation and bracket notation.

  9. DOM Manipulation

    Learn how to manipulate the Document Object Model (DOM) using JavaScript to dynamically update content, modify styles, and handle user interactions on web pages.

  10. Event Handling

    Master event handling in JavaScript, including attaching event listeners (addEventListener), handling user interactions (clicks, keypresses), and responding to events with callback functions.

  11. Asynchronous Programming (Promises)

    Understand asynchronous JavaScript and promises, including handling asynchronous operations, chaining promises, and using async and await for more readable asynchronous code.

  12. Fetch API (AJAX)

    Explore the Fetch API to make HTTP requests and retrieve data from external resources, enabling dynamic content loading without page refresh.

  13. Error Handling (try...catch)

    Learn error handling techniques in JavaScript using try...catch blocks to gracefully handle and manage exceptions that may occur during code execution.

  14. Local Storage

    Utilize the localStorage API in JavaScript to store and retrieve data persistently within the browser, enabling data caching and offline capabilities for web applications.

  15. ES6 Features (Arrow Functions)

    Discover ES6 (ECMAScript 2015) features in JavaScript, including arrow functions, template literals, destructuring assignments, and the let and const variable declarations.

Level 1:

HTML Basics

HTML stands for HyperText Markup Language, and it forms the backbone of web development. It defines the structure and content of web pages using a markup system with tags.

HTML documents are composed of elements structured using tags. Tags are enclosed in angle brackets < > and define different parts of a web page, such as headings, paragraphs, lists, images, etc.

There are headings from h1 to h6 and they keep decreasing in font-size. Usually the pages have only one h1.

Exercise Overview:

In this exercise you will create a basic HTML Document that includes one main heading, one subheading and a few paragraphs.

You may choose the content.

By completing this exercise, you will:

Understand the basic structure of an HTML document.

Gain familiarity with common HTML tags and their usage.

Learn how to create content hierarchy using headings.

Tips for Success:

Use Proper Nesting: Ensure tags are properly nested within each other head inside html, title inside head, etc.

Validate Your Code: Use online validators to check your HTML for syntax errors and compliance with standards.

Experiment: Don't hesitate to experiment with different tags and attributes to understand their behavior and use cases better.
